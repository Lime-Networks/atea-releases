# ATEA Releases

Publieke distributie van de Lime Networks Autotask Time Entry Assistent.

Dit is een automatisch bijgewerkte repository. Bestanden worden gepubliceerd via GitHub Actions vanuit de broncode repository.

| Bestand | Inhoud |
|---|---|
| `version.json` | Huidig versienummer |
| `extension.zip` | Laatste versie van de extensie |

## Installatie

Download `updater.ps1` uit de broncode repository en voer het uit.
